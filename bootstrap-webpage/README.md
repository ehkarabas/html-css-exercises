## Methodology

* Used

	* HTML elements
	* Bootstrap via local library

	* CSS selectors
	* CSS style attributes
	* Position properties

	* JavaScript

* Not used

	* Float, grid, flex layouts
	* SASS
	* CSS Media Queries
		* and so on...
